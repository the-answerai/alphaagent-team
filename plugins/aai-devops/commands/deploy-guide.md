# /deploy-guide - Deployment Guidance

Get deployment guidance tailored to your application and target platform.

## Usage

```
/deploy-guide
/deploy-guide to AWS
/deploy-guide blue-green strategy
```

## Workflow

1. **Analyze Application**
   - Application type and requirements
   - Current infrastructure
   - Scaling needs

2. **Recommend Strategy**
   - Blue-green deployment
   - Rolling deployment
   - Canary releases
   - Feature flags

3. **Infrastructure Guidance**
   - Platform recommendations
   - Resource sizing
   - Network configuration

4. **Deployment Process**
   - Pre-deployment checklist
   - Deployment steps
   - Verification procedures

5. **Rollback Planning**
   - Rollback triggers
   - Rollback procedures
   - Data migration considerations

## Output

- Deployment strategy recommendation
- Infrastructure diagram/description
- Step-by-step deployment guide
- Rollback procedures
- Monitoring recommendations
