# /research-blog - Research Blog Topic

Research a topic thoroughly before writing, including trends, examples, and what others have written.

## Usage

```
/research-blog <topic>
/research-blog "Next.js 14 App Router patterns"
/research-blog "microservices vs monolith"
```

## Research Areas

1. **Topic Deep Dive**
   - Core concepts and terminology
   - Recent developments and updates
   - Common misconceptions

2. **Codebase Analysis**
   - Find relevant examples in the codebase
   - Identify patterns being used
   - Note lessons learned

3. **Competitive Analysis**
   - What others have written on this topic
   - Gaps in existing content
   - Unique angles to explore

4. **Audience Research**
   - Common questions people ask
   - Pain points and challenges
   - Skill level considerations

5. **Resource Gathering**
   - Official documentation
   - Authoritative sources
   - Real-world case studies

## Output

- Topic summary
- Key points to cover
- Unique angle recommendations
- Code examples from codebase
- Resource links
- Content gaps to fill
