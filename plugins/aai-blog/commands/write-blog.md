# /write-blog - Write Technical Blog Post

Write a complete technical blog post with proper structure and engaging content.

## Usage

```
/write-blog [topic]
/write-blog "Getting Started with React Server Components"
/write-blog --outline outline.md
```

## Workflow

1. **Understand the Topic**
   - Clarify the target audience
   - Define the key takeaways
   - Determine technical depth

2. **Research & Gather Material**
   - Explore codebase for real examples
   - Research best practices
   - Identify common pitfalls

3. **Write the Post**
   - Engaging introduction with hook
   - Clear, structured body sections
   - Practical code examples
   - Actionable conclusion

4. **Technical Review**
   - Verify code examples work
   - Check technical accuracy
   - Ensure completeness

5. **Polish**
   - Add transitions
   - Improve readability
   - Suggest images/diagrams

## Output

- Complete blog post in markdown
- Code examples (tested)
- Suggested meta description
- Recommended tags/categories
