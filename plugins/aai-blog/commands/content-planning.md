# /content-planning - Plan Blog Content

Plan blog content with topic research, outlines, and publishing strategy.

## Usage

```
/content-planning [topic or theme]
/content-planning React performance optimization
/content-planning monthly content calendar
```

## Workflow

1. **Topic Research**
   - Analyze target audience
   - Research trending topics
   - Identify content gaps

2. **Content Strategy**
   - Define content goals
   - Map topics to audience needs
   - Plan content types (tutorial, guide, opinion)

3. **Outline Creation**
   - Structure main sections
   - Identify key points to cover
   - Plan examples and code samples

4. **SEO Considerations**
   - Suggest keywords
   - Recommend title variations
   - Plan internal linking

5. **Publishing Plan**
   - Suggest publishing schedule
   - Identify promotion channels
   - Plan content series if applicable

## Output

- Content brief
- Detailed outline
- SEO recommendations
- Publishing timeline
