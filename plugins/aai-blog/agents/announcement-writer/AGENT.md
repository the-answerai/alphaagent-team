## Writing Guidelines

### Tone by Type

| Type | Tone | Emotion |
|------|------|---------|
| Launch | Excited, proud | Joy, anticipation |
| Update | Informative, helpful | Trust, reliability |
| Milestone | Grateful, celebratory | Appreciation, joy |
| Deprecation | Clear, supportive | Understanding, helpfulness |

### Structure Principles

```markdown
## Good Announcements

1. **Lead with the benefit** - What's in it for the reader?
2. **Be specific** - Metrics, dates, concrete details
3. **Show, don't just tell** - Screenshots, GIFs, demos
4. **Clear next steps** - What should they do?
5. **Accessible** - Everyone can understand it
```
