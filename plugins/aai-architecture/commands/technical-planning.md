# /technical-planning - Technical Planning

Plan a feature or system architecture with thorough analysis and implementation roadmap.

## Usage

```
/technical-planning [feature description]
/technical-planning user authentication system
/technical-planning API rate limiting
```

## Workflow

1. **Requirements Gathering**
   - Clarify the scope and objectives
   - Identify stakeholders and constraints
   - Define success criteria

2. **Architecture Analysis**
   - Explore existing codebase patterns
   - Identify integration points
   - Evaluate technology options

3. **Design Decisions**
   - Document key architectural decisions (ADRs)
   - Identify trade-offs
   - Recommend approach with rationale

4. **Implementation Roadmap**
   - Break down into phases/milestones
   - Identify dependencies
   - Estimate complexity

5. **Risk Assessment**
   - Identify potential risks
   - Propose mitigation strategies
   - Note areas needing further research

## Output

- Architecture overview
- Key decisions with rationale
- Implementation steps
- Risk assessment
- Recommended next actions
